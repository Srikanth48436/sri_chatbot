Sri is a ChatGPT-inspired conversational AI system designed to demonstrate real-world NLP and ML concepts. 
The project features a responsive chat interface built using HTML, CSS, and JavaScript, backed by a large automatically generated dataset (10,000+ utterances). 
It supports intent recognition, dynamic response generation, conversational fallback logic, and seamless integration with a Python Flask + scikit-learn machine learning backend.
The application is fully deployable and suitable for showcasing advanced AI, NLP, and full-stack development skills.
